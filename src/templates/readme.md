# Templates Directory

This directory contains templates and assets for Docker and Unraid integration.

## Files

- `audiobook-tagger.xml`: Unraid Docker template
- `audiobook-tagger-icon.png`: Icon for Unraid Docker template (128x128 px recommended)

## Creating an Icon

1. Create a square PNG image (recommended size: 128x128 pixels)
2. Save it as `audiobook-tagger-icon.png` in this directory
3. The icon will be used in the Unraid Docker interface
